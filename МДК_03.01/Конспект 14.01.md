
# Спиридонова Софья 

## Что такое МДК 03.01?

*МДК* - Междисциплинарный курс. Это интегрированные знания из разных областей (алгоритмы, математика, теория БД, языки), собранные для решения одной практической задачи. 

*МДК 03.01* - Междисциплинарный курс "Моделирование и анализ ПО"

#### Цель освоения МДК - формирования у обучающих компетенций, необходимых для качественного освоения программной подготовки специалистов среднего звена.Некоторые из них:

-осуществлять ревьюирование программного кода в соответс. с тех. документицией;

-выполнять измерение характер. компонент ПП для опред. соответствия заданым критериям;

-исследовать созданный программный код с использование спец. программных средств для выявл. ошибок

-проводить сравнительный анализ ПП и средств разработки для выявления наилучшего решения согласно критериям,опред. ТЗ

*Зачем?*

Чтобы не просто изучать "предметы", а сразу видеть,как теория соьирается в рабочий навык: от написания кода до построения алгоритма до работы с даннами и исправление ошибок.

*Итог и Аттестация*

МДК 03.01 - Это теоретическая основа проф. модуля.После его изучения будет уч./произ. практика.Завершение модуля квалиф. экзаменом,где нужно показать знание теории и практическое умение.
